### Observações

- Os packs `CC0` podem ser usados sem atribuição obrigatória, embora manter o crédito continue sendo boa prática.
- Os packs `CC-BY` e `CC-BY-SA` exigem atribuição; `CC-BY-SA` também exige compartilhamento pela mesma licença para derivados distribuídos.
- Se você quiser, o próximo passo é eu selecionar só os arquivos de mesa/cadeira/monitor úteis e montar um catálogo enxuto já plugado no editor.
